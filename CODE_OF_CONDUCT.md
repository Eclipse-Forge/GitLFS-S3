# Code of Conduct

This project aims to maintain a respectful, safe, and collaborative environment for everyone.  
By participating in this repository, you agree to the following expectations.

---

## 💬 Expected Behaviour
- Be respectful and constructive in all discussions.
- Assume good intentions from others.
- Provide clear, actionable feedback.
- Keep communication professional and free of harassment.
- Respect differing viewpoints and technical approaches.

---

## 🚫 Unacceptable Behaviour
The following actions are not tolerated:
- Harassment, discrimination, or personal attacks.
- Malicious or intentionally harmful contributions.
- Spamming issues or PRs.
- Attempting to introduce security vulnerabilities.
- Bypassing contribution requirements (e.g., unsigned commits).

---

## 🛡 Our Rights
Project maintainers reserve the right to:
- Close or lock issues or PRs that violate this Code of Conduct.
- Reject contributions that do not meet project standards.
- Ban users engaging in malicious or harmful behaviour.
- Remove content that is inappropriate or disruptive.

These actions may be taken without prior warning if necessary to protect the project and its contributors.

---

## 📬 Reporting Issues
If you encounter behaviour or code that violates this Code of Conduct:
- Open a private GitHub security report (recommended for security issues), or
- Contact the maintainers directly via personal channels connected to their GitHub account

All reports will be reviewed and handled confidentially.

---

## 🤝 Commitment
By participating in this project, you help maintain a welcoming and productive environment for everyone.
